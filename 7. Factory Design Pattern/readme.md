### step 2: Concrete Products

### Step 3: Factory

```cs
public class VehicleFactory
{
    public static IVehicle GetVehicle(string type)
    {
        if (type.ToLower() == "car")
            return new Car();
        else if (type.ToLower() == "bike")
            return new Bike();
        else
            throw new Exception("Vehicle type not supported");
    }
}


```

### Step 4: Client Code

```cs
        vehicle2.Drive();
    }

```
